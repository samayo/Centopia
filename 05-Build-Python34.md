Install Python 3.4
========================

### install libraries
```
yum install gcc
```

download sorce and install. 
```
wget https://www.python.org/ftp/python/3.4.3/Python-3.4.3.tar.xz
tar xf Python-3.* 
$ cd Python-3.*
./configure --prefix=/opt/python3
$ make
$ make install
OR 
$ make altinstall
```
